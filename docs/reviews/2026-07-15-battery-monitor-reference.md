# BatteryMonitorのデザイン参照レビュー

- 日付：2026-07-15
- 参照実装：[zwnj/BatteryMonitor](https://github.com/zwnj/BatteryMonitor)
- 対象：通知領域から開く小型WPFフライアウトの外観と操作

## 結論

BatteryMonitorは、DisplayLightが目指す「通知領域から現れる小型コントロールセンター」に近い参照実装です。
特に、一枚の半透明サーフェス、小さい製品名、一つの主状態、境界線を抑えた内側カードという情報の強弱を採用します。

ただし、BatteryMonitorのWPF `Popup`、ホバー表示、ピン留め、ドラッグ移動は採用しません。
DisplayLightは設定変更を含むため、キーボードフォーカス、Narrator、高コントラスト、誤操作を避ける明示的な開閉を優先します。
実装方式は、標準タイトルバーとタスクバー表示を持たない小型WPF `Window`を維持します。

## 採用する要素

### 一枚の外側サーフェス

外枠はWindows 11の背景効果を使う一枚のサーフェスとして見せます。
内側の各領域へ同じ強さの枠線を繰り返さず、背景の明度差と余白でまとまりを示します。
背景効果を利用できない環境と高コントラストでは、不透明な単色サーフェスへ切り替えます。

### 主状態を一つに絞る情報階層

製品名は14 DIP程度の補助情報に下げます。
スリープ防止の状態を18 DIP程度の主表示とし、`オン`、`オフ`、`AC接続待ち`を文字で示します。
電源種別は「電源：AC接続」または「電源：バッテリー駆動」と表示し、現在側であることを補助的に示します。

### 境界線より余白を使うカード構成

外側余白は20 DIP、主要領域の間隔は16から20 DIP、カード内余白は14から16 DIPを基準にします。
ACとバッテリーは、それぞれ独立した弱いトーンのカードとして配置します。
展開中のカードだけに離散スライダーと未適用値を表示します。

### 現在値と変更後を区別する離散スライダー

スライダーは六つのプリセットを等間隔に置き、時間へ比例する連続目盛りとは扱いません。
トラックは2 DIP程度の細い線、未選択の目盛りは4から6 DIP、選択中のつまみは16 DIP程度を基準にします。

通常時は、適用済みの現在値と選択位置が一致するため、一つのつまみだけを表示します。
未適用の変更がある場合は、現在値の位置へ小さな輪郭リングを残し、変更後の位置をアクセント色のつまみで示します。
これにより、現在値からどこへ変更しようとしているかを、見出しの文言とスライダーの両方で確認できます。

フォーカスはつまみの外側へ2 DIPのリングとして表示し、つまみ自体の拡大でレイアウトを動かしません。
各プリセットには`1分`、`5分`、`10分`、`30分`、`60分`、`無期限`を常時表示します。
マウス操作は目盛りの選択とドラッグに対応し、キーボード操作は左右キーで一つずつ移動します。

### 控えめな状態変化

表示アニメーションを追加する場合は、短いフェードと4から8 DIPの移動に限定します。
Windowsのアニメーション設定が無効な場合は、アニメーションを行いません。

## 採用しない要素

- ホバーだけで開く操作
- 右Shiftキーによるショートカット
- ピン留めと自動クローズの切替
- フライアウトのドラッグ移動と位置保存
- ライトとダークの手動切替
- 大きな数値を中心に置くメトリクス表示
- 多数の小型メトリクスカード
- 絵文字を主アイコンとして使う表現
- 長い拡大縮小アニメーション
- WPF `Popup`への置換

これらはBatteryMonitorの監視用途には合いますが、DisplayLightのMVPに必要な設定項目を増やすか、明示的な操作とアクセシビリティを弱めます。

## DisplayLightへ反映する画面構成

```text
┌────────────────────────────┐
│ DisplayLight      電源: AC  ⋯│
│                            │
│ スリープ防止              │
│ オフ                  [切替]│
│ アイドルスリープだけを防止 │
│ AC接続時のみ          [ ]   │
│                            │
│ ディスプレイ消灯時間        │
│ ╭ AC電源時      現在 10分  ▴╮│
│ │ 変更後 30分               ││
│ │ 1 ─ ◌ ─ ● ─ ─ 無期限  適用││
│ ╰────────────────────────╯│
│ ╭ バッテリー時  現在 5分  ▾╮│
│ ╰────────────────────────╯│
└────────────────────────────┘
```

現在値は常にカード見出しへ表示します。
選択値が現在値と異なる場合だけ「変更後」と「適用」を表示し、適用後は両方を隠します。
展開ボタンは展開状態をAutomationへ公開し、矢印と読み上げ名を状態に合わせて更新します。

## 実装前に確認する項目

- Windows 11の一時ウィンドウ向け背景効果と単色フォールバックの見え方
- 半透明背景上での文字とフォーカス表示のコントラスト
- Narratorがスリープ防止をトグルとして読み上げること
- ACとバッテリーの展開状態を読み上げられること
- 360から392 DIPの範囲で、六つのプリセットが44 DIP程度の操作領域を保てること
- 現在値リング、変更後のつまみ、キーボードフォーカスを色だけに頼らず区別できること
- 非アクティブ化による自動クローズが補助メニューやNarrator操作を妨げないこと
